## Operadores de Comparação

### Igualdade 
- O retorno é sempre um valor booleano

```Python
saldo = 450
saque = 200

print(saldo == saque)

#False
```

### Diferença

```Python
saldo = 450
saque = 200

print(saldo != saque)

#True
```

### Maior que/ maior ou igual

```python
saldo = 450
saque = 200
print(saldo > saque)
#True
print(saldo >= saque)
#True
```

### Menor que/ menor ou igual

```python
saldo = 450
saque = 200
print(saldo <>> saque)
#False
print(saldo <>>= saque)
#False
```
